Класс - это шаблон для будующих объектов

Конструктор - это метод которы создает экземпляр класса

public - доступ во всем проекте
private - доступ только внутри класса
если не указывать - доступ только внутри пакета(packag)

геттер - выдает значение полей
сеттер - задают значение полям

extends - наследование, дает все поля и методы родительского класса, возможно только от одного класса
enum - данная сущность это перечисление
abstract - обязывает дочерние классы объявить этот метод
interface - своего рода контрат, которы он обязуется выполнить. Полей быть не может
implements - абстрактны класс

this. - ссылемся к объекту текущего класса
super. - вызов нонструктора и обращения к элементам родительского класса

Инкапсуляция - скрытие определенного участка кода
Наследвание - наследует поля и методы класса, за исключением приватных
Полиморфизм -  связан с переопределением, создание новой реализации у уже существующего метода ( + abstract)
